Research — Offline Bilingual Voice-Controlled Robot Arm

Advisor: Prof. KAWARAZAKI Noriyuki · Target: IEEE RO-MAN / SII / RA-L

Built a system that turns Thai/English voice commands into robot-arm actions: a local LLM (Qwen2.5 7B) decides whether to execute, ask for clarification, or reject each command, grounded by YOLOv8s vision and run on ROS 2. In live trials it reached 100% decision accuracy — matching a cloud model (Claude Sonnet 5) while running fully offline — and the experiments showed that clarification and vision are both necessary, and that a ~7B model is the threshold for reliable bilingual ambiguity handling.
