The Sundae Project

An AI RAG platform that answers customer questions from an organization's knowledge base across LINE and web chat, handing off to a human agent when needed.

View Repository

Overview

Sundae is a customer-support platform where an AI RAG chatbot answers questions from an organization's uploaded knowledge base and talks to customers over LINE and a website chat widget, handing the conversation to a human agent when needed. It is multi-tenant, so each organization manages its own bots, team, and knowledge base. Built as a team project with React, FastAPI, and Supabase.

My Contributions

  • Multi-tenant organization system — organization management, multi-admin and role-based access control, transfer ownership, member and bot counts, email invitations, and org logo upload.
  • Authentication & security — JWT auth, Row-Level Security policies, AES-GCM encryption for LINE secrets, the password-reset flow, and input validation.
  • Admin inbox & human handoff — a realtime inbox with status tracking and filters, cancel-handoff, and message source references.
  • Omnichannel integration — LINE webhook postbacks, an embeddable web chat widget, and the integration embed-code page.
  • Knowledge base — document upload (PDF/DOCX), storage display, and the PDF viewer.
  • Internationalization — a full Thai/English language toggle across every page.

The core RAG engine (Thai text splitting, vector search, embedding + reranker, and LLM generation) and the initial project scaffolding were built by a teammate.
